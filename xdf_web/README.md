# myvue

> 一个 Vue.js 项目

## 构建安装程序

``` bash
# 安装依赖
npm install

# 热加载服务 localhost:8080
npm run dev

# 构建压缩的生产版本
npm run build

# 构建生产并查看捆绑分析器报告
npm run build --report

# 运行单元测试
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```

详情查询 [向导](http://vuejs-templates.github.io/webpack/) and [vue-loader文档](http://vuejs.github.io/vue-loader).
